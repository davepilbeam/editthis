How to add/edit a News story

1. Prepare the Text

  • If uploading the text from a file then the file should be generated as plain text and saved in a UTF-8 Format** .txt file. Notepad's problems with Line Returns not appearing can usually be cured by pasting the text into Wordpad then cutting and pasting into the Page.
  • If created as a Word doc then we advise that you copy and paste rather than importing document.

2. Prepare any Images

  • Images should be JPEG, PNG or GIF formatted files, named with the appropriate correct extension (.jpg, .png, .gif), with a resolution of 72dpi.
  • Be aware that very large size images take longer to download and may affect the page display speed.
  • The webserver cannot process images larger than 10MB or exceed 2000px in width or 4000px in height.

3. Create the New Page

  •   Go to Pages >> News.
  •   Add a new Page

  • Fill in the Page information fields: (NB: in design terms shorter will display better on the site)
  • Page Title: Enter a Page Title which will become the title displayed on the Page itself.
  • Page URL: Enter a Page URL (maximum: 60 characters) which will become the URL slug of the HTML page (eg: My-New-Page.html) and be displayed on the crumbline (eg: >> My New Page).
  • Short Title: Enter a Short Title (maximum: 30 characters) for use instead of the full Page Title when there is limited space such as in Tablet or Mobile view.
  • Page based on: Select an existing Page in the dropdown menu to base your new Page on. Choose a Page that is either the same or the closest in appearance to the format you require.

  •   Save the new Page and return to the News Pages list.
  •   Your new Page should appear listed in blue; it will not be visible on the site until you click the show page icon after you have finished editing.

4. Edit the New Page

  •  Open the Editing area for the new Page.
  •  Choose Edit this Page

  • 4.1 Replace Image
  • Click on the green editable area for the Image you wish to edit.
  •   Select the Upload icon.

  • Drag your image into the box with the dashed line border.

  • Check the Generate Sizes box.
  •   Upload the image.
  •  Go to Image Editing.
  • Choose your Image from the list.
  •  Update the Image.

  • 4.2 Replace Text
  • Click on the green editable area for the Text you wish to edit.
  •  Choose Paste.

  • Drag your text file or paste your new text into the area.

  •  Update the Text.
  •   Save the updated Page.

**Saving a Text File in UTF-8 Format:

In Notepad: Go to File -> Save As and choose "Save As Type: Text Documents (*.txt)" and "Encoding: UTF-8".

NB: Notepad does not preserve linebreaks when pasting directly into some browsers - if required first paste your document into Wordpad, then cut from there.

In Wordpad: Go to Save As -> Other Formats -> Save As Type and choose "Unicode Text Format (*.txt)". Select the "Save in this format by default" checkbox to retain this setting for the future.

In Microsoft Word: Go to File -> Save As and choose "Text Only (*.txt)".
