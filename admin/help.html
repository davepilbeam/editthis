    Editing text overview

    Editing text overview

    Editing text through the content management system can be done manually or by dragging in a UTF-8 formatted file or by copying and pasting from Notepad, Wordpad or Word. Please see detailed instructions in the 'Saving a Text File in UTF-8 Format' section further down the page.

    How to edit text

    How to edit text

    Locate the page you wish to amend and click on the   edit icon.

    You will then be presented with the Edit overview menu for that page where there are various options.

    To edit the content text on the page click on the  edit icon.

    You will now see a version of the live web page with overlayed green editable areas.

    Please note: not all the green areas are currently editable e.g feeds and modules, development of these CMS tools is in progress and will be released for public use in the near future.

    Click on the text area you wish to amend/update

    Small icons will appear in the top right of the editable area within the green dashed border line.

    Click on the   edit icon. You can now edit the text manually or paste text in.

    Clicking on the text input icon will allow you to upload or drag in new copy  

    Selecting a paragraph

    You can select a whole paragraph to embolden, italicise, change text format or to create a link by clicking once anywhere on the paragraph - a light blue highlight will indicate you have done this.

    Selecting a string of text

    You can select a string/portion of text by clicking and dragging as you would in most other software.

    The selected text will be highlighted in blue.

    You can now embolden, italicise, change text format or to create a link of the highlighted text by using the icons in the toolbar.

    Toolbar text edit icons

    To embolden highlighted text click on the   icon

    To italicise highlighted text click on the   icon

    To change the highlighted text format click on the   icon, click multiple times to carousel through the options (these include bullets and sub-bullet options).

    To create a link from the highlighted text click on the   icon. This will then take you to the link menu window.

    Pasting in text

    You can paste text from Word, Notepad(PC), Wordpad(PC) & TextEdit(Mac)

    Uploading a text file

    Please note uploading a text file will replace all text within the editable area regardless of having any text highlighted within the editable area, prior to uploading.

    Once you have prepared your text file (see Saving a Text File in UTF-8 Format further down the page)

    You can upload new/replacement text by clicking on the text input icon  

    To drag a text file onto the editable text area you must click the   edit icon before dragging the file in.

    Saving the text changes

    You can save any text changes made by clicking on the  update icon. You can now edit other areas of the page or save and publish the page.

    Please note: If you click off the editable area before you have clicked the update icon then all changes to that editable area will be lost.

    To save the page click the large  save icon in the top right of your browser window.

    Editing images overview

    Editing images overview

    Editing images through the content management system lets you either upload a new image or replace an image with another that already exists on the CMS.

    Preparing new images

    Preparing new images

    Images should be JPEG, PNG or GIF formatted files, named with the appropriate correct extension (.jpg, .png, .gif), with a resolution of 72dpi.

    Be aware that very large size images take longer to download and may affect the page display speed.

    The webserver cannot process images larger than 10MB or exceeding 2000px in width or 4000px in height.

    How to edit images

    How to edit images

    Locate the page you wish to amend and click on the   edit icon.

    You will then be presented with the Edit overview menu for that page where there are various options.

    To edit an image on the page click on the  edit icon.

    You will now see a version of the live web page with overlayed green editable areas.

    Please note: not all the green areas are currently editable e.g feeds and modules, development of these CMS tools is in progress and will be released for public use in the near future.

    Small icons will appear in the top right of the editable image area within the green dashed border line.

    To replace the image with another image currently on the CMS

    Click on the   edit icon.

    You will then be presented with the Edit Image page and all the image options available from the same folder as the image you wish to replace.

    We recommend you use only the images from the same folder, as these will be sized correctly for similar usage.

    To upload a new image.

    Please note: the CMS has the ability to crop images to a specific sizes/proportions for specific uses on the website.

    Click on the   upload icon.

    You will then be presented with the Upload Image page where you can specify the type of image you want your upload to be so that the CMS can crop/proportion the image appropriately.

    Once you have selected the type of image you can click on the upload area below to find your file or drag an image file into the box.

    Please note: You can upload multiple images at the same time.

    A thumbnail of the image(s) and the name of the image file(s) you have selected will appear below the upload box.

    You can click the   exit icon to clear any image(s) you don't want to upload.

    Clicking the large   save icon in the top right of your browser window will begin the upload process and save the image(s).

    You will be taken back the Edit Image page where you will be able to see the new images that have been uploaded amongst the existing images in that folder.

    New images and images that have not been used on a live page have a light blue background.

    Clicking on an image will show a preview of the image on the left side.

    To save the image selection click the large   update icon in the top right of your browser window.

    You will then be taken back to the Edit Page where you can see the new image in place.

    You will then need to click the large   update icon in the top right of your browser window to save any image changes.

    Creating a new page

    This website has a variety of page templates for specific sections.

    Prepare any Images

    Prepare any Images

    Create the New Page

    Create the New Page

      Find the section/area where you want the new page to go.

      Go to Add a new Page

    Fill in the Page information fields: (shorter text will look better when the Page is displayed)

    Enter Page Title (maximum: 150 characters) which will become the title displayed on the Page itself.

    Enter Title of Page in Menus (maximum: 50 characters) which will become the Page's entry in the site Menus as well as the URL of the HTML page displayed in the browser, eg: My New Page and My-New-Page.html.

    Enter Page Short Title (maximum: 30 characters) which will be displayed instead of the full Page Title when required in tablet or mobile view.

    You can either use an existing existing Page in the Base Page on dropdown menu to base your new Page on, or use one of the default page templates that come with the site.

      Save the new Page.

    Edit the New Page

    Edit the New Page

    After creating the new page you will then see the Page overview options for the newly created page

      Please note: Initially your Page will be hidden in the Site Map and News Index Page. When you have completed editing your new Page, you can return here to unhide your Page.

      Select the Edit Page Content icon for the new Page.

    Replace Text

    Click on the green editable area for the Text you wish to edit.

    Unhide the Completed Page

    Unhide the Completed Page

    Returning to the Pages list, your new hidden Page should appear listed at the bottom.

      Select the Show in Sitemap icon to make it appear in the Sitemap, this will also enable the page to be visible if it is part of a group of pages that is dynamically pulled into other parent pages e.g. a product page (UPM Fine) is pulled into the product index page (Uncoated Papers).

      Select the Show in Menus icon to make it appear in the main navigation menu for the website.

    You can change existing page titles and menu links (URLs) using the Page overview options

    We recommend keeping the Page Title and the Title Shown in Menus the same to avoid any confusion.

    Changing a Page Title

    Changing a Page Title

    The following steps will change the title displayed on the page itself in a H1 tag.

    From the CMS Home page find the page you wish to amend.

      Select the Edit icon.

    You will then see the Page overview options for the page

      Select Rename Page Title

    Enter/update the Page title in the field provided.

    It is also advisable to update the Page Short Title at this time. This can be edited by clicking on the Edit Page Short Title option further down the overview page.

    Changing the Title Shown in Menus

    Changing the Title Shown in Menus

    The following steps will change the Page name displayed in the Site Navigation Menus and will also be the actual Page URL, eg: Menu name My New Page will have the Page URL My-New-Page.html where spaces are replaced by dashes and only basic characters are allowed.

    From the CMS Home page find the page you wish to amend.

      Select the Edit icon.

    You will then see the Page overview options for the page

      Select Title Shown in Menus

    Enter/update the new page link title in the field provided.

    Saving changes

    Saving changes

     Click to save (bottom left in the overview page) when you have completed all changes required.

    Saving a Text File in UTF-8 Format

    Windows Notepad

    In Notepad go to File -> Save As and choose "Save As Type: Text Documents (*.txt)>" and "Encoding: UTF-8".

    Note that Notepad does not preserve linebreaks when pasting directly into some browsers - if required first paste your document from Notepad into Wordpad, then cut from there.

    Windows Wordpad

    Windows Wordpad

    In Wordpad: Go to Save As -> Other Formats -> Save As Type and choose "Unicode Text Format (*.txt)". Select the "Save in this format by default" checkbox to retain this setting for the future.

    Microsoft Word

    Microsoft Word

    In Microsoft Word: If your text is in a Word document you should copy directly from the Word document then paste directly into an editable text area on a Page by holding down the CONTROL+SHIFT+V keys to remove the extra Word formating.

    Altering Top-Level Pages on the Site

    Adding, Deleting or Re-ordering a Top-Level Navigation Entry

    It is not advisable to add, delete or re-order top-level Pages in the main Navigation Bar unless you have full Administrator access. This is because the Site's CSS Stylesheets must also be altered to accomodate the new display and this requires specialist knowledge. This will become apparent if the new Navigation Bar does not fit the screen, or the Navigation Lists in the Site's Footer Area become mis-aligned.

    Create your Page as usual, but keep it hidden from the Menu.

    When it is complete, email admin@thatsthat.co.uk telling us which Page you have created and specify where it is to be inserted. We will then adjust the CSS display and make the Page visible for you.
